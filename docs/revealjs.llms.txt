# reveal.js For vibe-coding-slides

> Curated reveal.js reference for migrating this repository's custom slide runtime to `reveal.js`, preferably through `@revealjs/react`, while keeping Astro routing, course metadata, bilingual pages, and lecture content structure intact.

Important notes:

- This repository is a lecture-material website, not a general app. Prefer presentation features that improve teaching flow, readability, speaker support, and shareability.
- Prefer `@revealjs/react` over manual React initialization unless there is a clear integration blocker.
- Keep Astro routes and course metadata as they are. Replace only the slide deck runtime.
- Do not re-implement built-in reveal.js features such as keyboard navigation, touch navigation, slide numbers, overview mode, speaker notes, PDF export, fragments, or scroll view unless repo-specific behavior truly requires it.
- For scrollable content inside slides, use `data-prevent-swipe` to avoid conflicts with touch navigation.
- For npm imports, use top-level package paths such as `reveal.js/plugin/markdown`; avoid old `dist/` import paths from pre-6.0 examples.
- `plugins` in `@revealjs/react` are initialization-time only. Dynamic plugin swapping after mount should not be assumed.
- `view: 'scroll'` is especially relevant for this repository because lectures may need both a presenter-friendly slide mode and a reader-friendly linear mode.
