# Cusdis 자동화 예제

이 폴더는 강좌 상세강의자료에서 연결되는 실습 자산을 모아 둔 곳입니다.

## 추천 순서

1. `sample-comment-webhook.json`
2. `gemini-moderation-prompt.ko.md`
3. `normalize-gemini-output.js`
4. `approve-comment-request.json`
5. `ops-checklist.ko.md`

## 이 파일들로 보는 것

- webhook 샘플로 Cusdis 입력 구조를 먼저 확인합니다.
- 프롬프트 문서로 Gemini가 어떤 JSON 계약을 지켜야 하는지 봅니다.
- JavaScript 예제로 AI 응답을 안전하게 정리하는 법을 익힙니다.
- 승인 요청 예제로 최종 API 호출 형태를 확인합니다.
- 체크리스트로 배포 전후 점검 포인트를 정리합니다.

## 권장 실습
