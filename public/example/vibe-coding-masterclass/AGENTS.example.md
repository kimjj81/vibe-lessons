# Example AGENTS.md

## Role

The assistant is responsible for small, verifiable product changes inside this repository.

## Working rules

- Prefer minimal diffs.
- Confirm acceptance criteria before broad changes.
- Run the narrowest meaningful verification first.
- Report changed files, commands, and remaining risks.

## Guardrails

- Do not add dependencies without approval.
- Do not change deployment behavior silently.
- Do not claim success without test or build evidence.
