## 왜 이 구성이 필요한가

이 강좌의 핵심은 한 번의 마법 프롬프트가 아닙니다. 반복 가능한 구조를 만드는 것입니다.

- 제품 의도 정리
- 작업 규칙 문서화
- 기술 선택 기준 정리
- 단순한 아키텍처 스케치
